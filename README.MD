#MEMBER ASSISTANCE APPLICATION

## DEPENDENCIES
Install Java 8 (openjdk/jre)
Install git 1.9.x up
Install Gradle 6.x up

## SECURITY APP
Download and install keycloak 12.x
Need to import realm json file setup/(member-assistance-realm.json).

Start app on ubuntu:
sudo ./standalone.sh -Djboss.socket.binding.port-offset=100
